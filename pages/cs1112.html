Today we discussed the Fibonnaci sequence 1 1 2 3 5 8 etc and showed numerically that the ratio of terms in the sequence, \(f_{n+1}/f_n\), converged to the golden mean: \( (1+\sqrt{5})/(2) \). The code looks something like this: 


    f_old = 0; f_cur = 1; n = 1;
    % f_cur is always the nth Fibonacci number
    % f_cur + f_old is always the n+1st Fibonacci number.
    while ( abs( (f_cur+f_old)/f_cur  - (1+sqrt(5))/2 ) >.000001 )
        % Update: 
        f_new = f_old + f_cur; 
        f_old = f_cur; 
        f_cur = f_new; 
        n = n+1;
    end
    fprintf('golden mean: %8.5f', (f_cur+f_old)/f_cur)
